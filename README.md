# Кто здесь дракон?

**Смелый рыцарь грубо нарушает закон, вбегая на частную собственность дракона, и крадёт принцессу у крылатого чудовища.**

На самом деле, принцессу дракону подарили на день рождения (со всеми необходимыми нотариальными согласиями), а гора является его полностью легальным жилищем, о чём красноречиво свидетельствует табличка «Посторонним В.» на двери в пещеру, написанная сносно читаемым почерком для драконьей лапы. Игнорирование этой таблички — ваша первая, но не последняя ошибка.

**Игра, настойчиво рекомендующая читать пользовательские соглашения, предостережения на батарейках и, особенно, вывески на дверях неевклидовых пещер.**
